# 测试策略文档

## 测试金字塔

| 层级 | 比例 | 工具 |
|------|------|------|
| 单元测试 | 70% | pytest / Jest |
| 集成测试 | 20% | pytest-asyncio / MSW |
| E2E 测试 | 10% | Playwright |

## 后端测试

- **单元测试**: 领域实体、服务逻辑、工具函数
- **集成测试**: API 端点、数据库交互
- **E2E 测试**: 完整用户流程

## 前端测试

- **单元测试**: 组件渲染、Hook 逻辑、工具函数
- **集成测试**: API 集成、页面交互
- **E2E 测试**: 用户流程、跨浏览器

## 覆盖率目标

- 后端: ≥ 85%
- 前端: ≥ 80%

## CI/CD 集成

1. 代码质量检查
2. 单元测试
3. 集成测试
4. 端到端测试
